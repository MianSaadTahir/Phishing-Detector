# Threat Guard

**Threat Guard** is a cyber safety web app built using Python's Flask framework. It integrates real-time scanning of URLs, QR codea and files using the VirusTotal API, checks for compromised passwords via HaveIBeenPwned, blocks DDoS attempts through rate limiting, and stores phishing-related data securely using SQLite databases.

## Features

- **URL and File Scanning**: Scan links, QR code and uploaded files for malware and phishing threats.
- **Pawned Detection**: Uses HaveIBeenPwned API to detect breached emails and passwords.
- **User Authentication**: Secure sign-up/login system with hashed passwords.
- **Database Architecture**: SQLite used for storing users, phishing URLs, IPs, and breach data.
